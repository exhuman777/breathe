The Problem
"I shipped more code last quarter than any quarter in my career. But I burned out. I didn't quit or have a breakdown, but just stopped caring."
Siddhant Khare, Software Engineer — covered by Business Insider, Techmeme

Something broke in the developer experience. AI writes code faster than we can understand it. The bottleneck moved from creation to comprehension. From typing to being present. You used to build. Now you review, validate, decide, re-prompt, repeat. Creating is energizing. Reviewing is draining.

The Solution
inhale
breAIth is running. Claude is building.

breAIth is a CLI meditation companion that hooks into Claude Code. When Claude starts building, breAIth activates: guided breathing with 3D wireframe geometry. When Claude finishes or needs your input, a gong sounds. Your meditation is exactly as long as Claude's thinking. No timers. No apps. No scheduling. The AI creates the space. You fill it with presence.

Install
recommended: clone + run
git clone https://github.com/exhuman777/breathe.git ~/breaith
cd ~/breaith && python3 breathe.py
install claude code hooks (auto-start on builds)
cd ~/breaith && python3 breathe.py --install-hooks --confirm
or: one-liner (download only)
curl -fsSL https://raw.githubusercontent.com/exhuman777/breathe/main/breathe.py -o ~/breaith.py && python3 ~/breaith.py
already installed? just run
python3 ~/breaith/breathe.py

Requires Python 3.6+ and a terminal with Unicode support. Zero dependencies. macOS recommended (system sounds + Claude Code hooks).

Why breAIth Exists

The psychology of "doing mode" vs. "being mode" is directly relevant. Doing mode: constant activity, goal-oriented, planning, evaluating current vs. desired state. Being mode: present-moment awareness without judgment. Modern AI tools hyper-optimize for doing mode. Most developers spend zero time in being mode. breAIth is the bridge.

This is not a productivity hack. Not "meditate to code better." The code will be fine. Claude handles that. This is about what happens to you while the machine works. Do you fragment your attention across 12 tabs? Or do you breathe? The AI age demands a new skill: the ability to be still when everything around you accelerates.

"The real question isn't what's left to do. It's what matters most to do now."
The answer might be: nothing. Just breathe.
